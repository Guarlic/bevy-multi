# Bevy Multi

- 베비 엔진으로 만든 멀티플레이 게임의 테스트.
- 2인용, P1 (오른쪽) 은 K, P2 (왼쪽) 은 C 키를 눌러 탄막 발사 가능
- 상대 탄막에 닿으면 패배.
